# Hard Copy Public Library Folder Requirements

The public Hunt Library wall should be curated and should not expose every runtime database file.

Visible public folders:

1. UTAH DWR RULES & REGULATIONS
   - Current-year hunt proclamations, field guides, guidebooks, and application materials only.

2. HARVEST DATA
   - Public harvest data may show available years.

3. DRAW RESULTS
   - Public draw results may show available years.

4. CONSERVATION PERMITS
   - Current-cycle/current-year conservation permit information only.

5. HUNT EXPO
   - Current-year permit numbers only.

6. SIGNIFICANT DATES / CALENDAR
   - Application windows, opening dates, deadlines, season calendars, and related date references.

7. 2026 HUNT UNITS / PERMIT NUMBERS
   - Current-year hunt code, hunt name, hunt unit, and posted/allotted permit number data only.

8. UTAH OUTFITTERS BY HUNT CODE/HUNT NAME
   - Outfitter resources tied to hunt code and hunt name.

Runtime prediction files such as point ladders, draw reality engine outputs, enriched hunt master tables, and coverage/audit files remain part of the hard-data system and prediction engine. They should not clutter the public wall unless intentionally exposed as a summarized public file.
